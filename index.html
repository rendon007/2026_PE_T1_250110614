Clasificación de datos:

1. Datos Cuantitativos Discretos

Definición: Son datos numéricos que resultan de un conteo. Solo pueden tomar valores enteros exactos y finitos.

Al lanzar un dado, solo puede salir 1, 2, 3, 4, 5 o 6. No puede salir "3.5".

Ejemplo seleccionado: Lanzamiento de un dado (100 veces).

2. Datos Cuantitativos Continuos

Definición: Son datos numéricos que resultan de una medición. Pueden tener decimales infinitos dentro de un rango.

La altura no es exacta; alguien puede medir 1.70m, 1.705m o 1.7058m dependiendo de la precisión.

Ejemplo seleccionado: Estatura de estudiantes (1.50m - 1.95m).

3. Datos Cualitativos (Categóricos)

Definición: Se refieren a cualidades o marcas que no son números matemáticos.

Sirven para agrupar elementos por características comunes.

Ejemplo seleccionado: Marca de Teléfono Móvil.
